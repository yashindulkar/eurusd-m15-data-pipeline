# Validation Reports

Run `python3 scripts/validate_data.py` to generate:

- `EURUSD_M15_quality_report.md`
- `EURUSD_M15_missing_candles.csv`
- `EURUSD_M15_spikes.csv`

The Markdown report is tracked as the compact quality summary. Large generated CSV diagnostics are intentionally not tracked in git.
